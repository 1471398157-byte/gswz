# 公司网站

现代化的公司介绍网站

## 部署方式

### 1. Vercel 部署
1. 访问 https://vercel.com
2. 使用 GitHub 账号登录
3. 导入此仓库
4. 自动部署完成

### 2. Cloudflare Pages 部署
1. 访问 https://pages.cloudflare.com
2. 连接 GitHub 账号
3. 选择此仓库
4. 部署完成

### 3. GitHub Pages 部署
1. 进入仓库 Settings
2. 找到 Pages 选项
3. 选择 main 分支
4. 保存即可

## 本地预览

直接在浏览器中打开 `index.html` 文件即可。
